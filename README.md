## Getting Started

* Clone this repo and get into the drugman folder

```
cd Investocarcy
```

* Update config/key with
1. MongoURI
2. JWT secret
3. Sendgrid key

* Install Packages

```
npm install
cd config
npm install
```

* Start Servers

```
npm run dev
```
